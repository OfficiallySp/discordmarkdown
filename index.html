Markdown Cheat Sheet

  • Bold: **text**
  • Italic: *text*
  • Underline: __text__
  • Strikethrough: ~~text~~
  • Inline Code: `code`
  • Code Block: ```code```
  • Quote: > text
  • Multi-line Quote: >>> text
  • Spoiler: ||text||
  • Header 1: # text
  • Header 2: ## text
  • Header 3: ### text
  • Subtext: -# text
  • Bullet List: * item
  • Masked Link: [text](url)
